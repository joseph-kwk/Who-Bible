# Quick Feature Testing Guide

## üéØ How to Test All Features

### 1. Theme Testing
1. **Current Theme**: Check if you're in dark or light mode
2. **Toggle Theme**: Click the sun/moon icon in the top-right
3. **Verify**: Background should change from dark charcoal to beautiful sky blue with grey accents
4. **Eye Comfort**: Light theme should feature elegant sky blue gradients with floating cloud patterns

### 2. Game Mode Testing

#### Solo Mode
1. Click "Solo Mode" button
2. Adjust settings (difficulty, number of questions)
3. Start quiz and answer questions
4. Check scoring and progress tracking

#### Timed Mode  
1. Click "Timed Mode" button
2. Set time limit (e.g., 60 seconds)
3. Start quiz and watch countdown timer
4. Verify time pressure affects gameplay

#### Challenge Mode
1. Click "Challenge Mode" button  
2. Enter two player names
3. Start game and verify alternating turns
4. Check that scores track separately

#### Study Mode
1. Click "Study Mode" button
2. Browse Bible character information
3. Test search functionality
4. Try filtering options

### 3. Settings & Data Testing
1. **Language**: Try switching between English, French, Spanish
2. **Export**: Click "Export JSON" to download data
3. **Import**: Import a custom JSON file
4. **Reset**: Reset to default Bible people data

### 4. Accessibility Testing
1. **Keyboard**: Use Tab, Enter, Arrow keys to navigate
2. **Focus**: Check visible focus indicators
3. **Screen Reader**: Test with assistive technology if available

### 5. Mobile Testing
1. Open on mobile device or narrow browser window
2. Check responsive design works
3. Verify touch interactions
